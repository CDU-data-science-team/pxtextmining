Using a trained model

The results folders (e.g. results_label) always contain a fully trained model in .sav format, as well as performance metrics for the model. The models saved here are used to make predictions on the experiencesdashboard frontend.

To use one of these pretrained models to make predictions:

  1. Prepare a pandas DataFrame of the text you want to classify.
  2. Preprocess the text using the pxtextmining.factories.factory_data_load_and_split.clean_data function, specifying target=False
  3. Use this dataframe as the dataset argument in factory_predict_unlabelled_text.

An example of these steps in action is available in execution_predict.

Be wary of preparing your dataset using Excel, as it can cause issues due to text encoding errors. LibreOffice is a good alternative.
